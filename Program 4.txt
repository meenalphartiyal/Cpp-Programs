
Program 4: Write a program which arrange the elements of the sorted array 
           such that first element should be maximum value, second minimum 
           value, third second maximum value, fourth second minimum value and so on.
For example
     Input : arr[] = {1, 2, 3, 4, 5, 6, 7}
     Output : arr[] = {7, 1, 6, 2, 5, 3, 4}
